# SQL Data Analyst Practice

## Project Overview

This project contains a structured SQL dataset and a collection of SQL queries designed to practice and demonstrate core SQL skills required for a Data Analyst role.

The queries range from **basic SQL operations to advanced analytical problems**, helping users strengthen their understanding of relational databases and analytical querying.

---

## Dataset Description

The project uses three relational tables:

### Employees

Stores employee information.

| Column        | Description                 |
| ------------- | --------------------------- |
| employee_id   | Unique ID for each employee |
| name          | Employee name               |
| department_id | Department identifier       |
| salary        | Employee salary             |
| hire_date     | Date the employee joined    |

---

### Departments

Stores department details.

| Column          | Description            |
| --------------- | ---------------------- |
| department_id   | Unique department ID   |
| department_name | Name of the department |

---

### Projects

Stores project assignments and budgets.

| Column       | Description                      |
| ------------ | -------------------------------- |
| project_id   | Unique project ID                |
| employee_id  | Employee assigned to the project |
| project_name | Name of the project              |
| budget       | Budget allocated to the project  |

---

## SQL Concepts Covered

The project includes queries related to:

* Data retrieval using SELECT
* Filtering data with WHERE
* Aggregations using COUNT, SUM, AVG, MAX
* GROUP BY and HAVING
* Table joins
* Subqueries
* Window functions
* Ranking functions
* Business-style analytical queries
